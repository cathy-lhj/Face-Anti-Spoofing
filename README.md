人脸识别技术已广泛应用于金融、门禁和政务等领域，但现有系统仍容易
受到打印照片、视频重放等欺骗攻击。同时，部分人脸反欺诈方法还存在对光照变化敏
感、跨场景泛化能力有限以及模型参数量较大的问题。针对上述情况，本文提出一种基
于注意力机制的双流卷积神经网络人脸反欺诈方法。该方法以MobileNetV3作为轻量化
骨干网络，构建由RGB原始图像与MSR多尺度Retinex增强图像组成的双流输入结构，
分别学习人脸纹理细节特征和受光照影响较小的高频特征，并实现两类特征的融合。为
提高关键区域特征提取效果，本文设计了注意力的特征融合方法，用于增强有效信息、
减少冗余特征影响。同时采用二分类交叉熵损失函数，并结合RAdam与Lookahead优
化器，以提高模型训练过程的稳定性和泛化能力。我们在 CASIA-FASD、REPLAY
ATTACK 和OULU-NPU等多个公开数据库上对所提出的框架进行了评估，取得了极具
竞争力的性能表现。为进一步验证所提策略的泛化能力，我们开展了跨数据库实验和消
融实验，实验结果表明本文方法具备较好的有效性。在此基础上，本文进一步设计并实
现了实时人脸反欺诈系统，包括视频采集、人脸预处理、双流网络检测和结果可视化四
个模块，该系统在模型推理的基础上，设计并实现了一种基于滑动窗口的多帧投票融合
算法，提高了系统稳健性，可在复杂光照环境下完成真实人脸与欺骗人脸的判别，具有
一定的实际应用价值。
